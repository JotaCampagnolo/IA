Todas a bibliotecas utilizadas neste Trabalho podem ser obtidas através do PIP do PYTHON.
    - Para instalar a biblioteca NUMPY:
        $ pip install numpy

    - Para instalar a biblioteca MATPLOTLIB:
        $ pip install matplotlib

    - Para instalar a biblioteca GYM:
        $ pip install gym

    - As bibliotecas RANDOM e OS.PATH já estão presentes no SO Linux.

Para executar o Programa, basta navegar até o diretório do Trabalho e utilizar o seguinte comando:
    $ python cartpole.py

    - Nota: Existem 3 opções que podem ser habilitadas e desabilitadas no código fonte:
        LOAD_DICT = <True ou False>: Habilite para carregar os estados previamente salvos.
        PLOT_ALL_EPS = <True ou False>: Habilite para exibir graficamente todos os Episódios.
        PRINT_DICT = <True ou False>: Habilite para printar a configuração final do Dicionário de Estados.
